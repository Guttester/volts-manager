## 🚀 Volt’s Manager

Sistema de gestão de produtos desenvolvido para simular um ambiente real de produção, aplicando práticas profissionais de desenvolvimento, qualidade e automação.

O projeto utiliza uma stack moderna com:

![TailwindCSS](https://img.shields.io/badge/Tailwind-38B2AC?logo=tailwind-css&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-336791?logo=postgresql&logoColor=white)
![Cypress](https://img.shields.io/badge/Cypress-17202C?logo=cypress&logoColor=white)

![GitHub Actions](https://img.shields.io/badge/GitHub%20Actions-6e6e6e?logo=github-actions&logoColor=white)

- PostgreSQL (banco de dados)
- Java (backend / API)
- Angular + TailwindCSS (frontend)
- Cypress e Selenium (testes automatizados)
- Docker (conteinerização)
- CI/CD (pipeline automatizado)
- Figma (Design UIX)

O objetivo é demonstrar na prática um fluxo completo de desenvolvimento, desde a construção da aplicação até a validação e automação do ambiente.

---

## 📦 Estrutura do Projeto

O projeto é dividido em módulos, cada um com responsabilidade específica.

- **volts-manager-back** → API (backend em Java)
- **volts-manager-ui** → Interface (Angular + TailwindCSS)
- **volts-manager-tests** → Testes automatizados (Cypress / Selenium)

Cada módulo possui um README próprio com instruções de instalação, execução local, dependências e comandos principais.
